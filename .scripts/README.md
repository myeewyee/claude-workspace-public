# Scripts

Utility scripts for the Claude workspace. Update this file when creating or archiving scripts.

Spent or superseded scripts go in `archive/`.

## Infrastructure

- `rename-links.ps1` : Vault-wide wiki-link rename after file moves. Supports single rename, batch via manifest, dry-run/execute.
- `detect-session-id.sh` : Extract current Claude Code session ID from process list.
- `Launch-Obsidian.ps1` : Launch Obsidian with clean workspace.json (prevents slow startup).
- `fix-claude-extension.ps1` : Patch Claude Code VS Code extension for Windows activation.
- `asana.py` : Read-only Asana workspace queries. Full CLI with subcommands (me, projects, task, search, etc.). Requires `ASANA_PAT`.

## Travel

- `flight-search.py` : Single flight search via Kiwi MCP endpoint. Full argparse CLI.
- `accommodation-search.py` : Search Booking.com + Airbnb via Apify actors. Full argparse CLI. Requires `APIFY_API_TOKEN`.
- `car-rental-search.py` : Search Kayak car rentals via Apify actor. Full argparse CLI. Requires `APIFY_API_TOKEN`.
- `review_scraper.py` : Pull hotel reviews from Google Maps + Booking.com via Apify. Requires `APIFY_API_TOKEN`.
- `air-quality-search.py` : Real-time AQI comparison + historical monthly PM2.5 trends. Dual-source: WAQI (current) + OpenAQ (history). Requires `WAQI_TOKEN` + `OPENAQ_API_KEY`.

## Shopping

- `lazada-search.py` : Search Lazada Thailand products via Apify actor. Full argparse CLI (keyword, sort, price range, rating filter). Requires `APIFY_API_TOKEN`.

## YouTube

- `youtube-browse.py` : Browse YouTube channel videos via Data API v3. Lists recent videos with title, date, views, duration. Full argparse CLI. Requires `YOUTUBE_API_KEY`.

## Vault utilities

- `find_hubs.ps1` : High-connectivity note analysis (top 50 by backlink count).
- `search_sessions.py` : CLI wrapper for Vault Intuition MCP session search. Requires `mcp` package.
- `get_session_detail.py` : CLI wrapper for Vault Intuition MCP session detail. Requires `mcp` package.
- `list-vault-notes.ps1` : List notes per vault directory with sample filenames.
- `vault-count.ps1` : Vault structure analytics (journal breakdown, MOC list, permanent notes).

## Session/debug

- `check-session.ps1` : Session JSONL log inspector. Reports event types and tool calls.
- `extract-timestamps.ps1` : Extract timestamps from session JSONL logs.

## Data files

- `renames.json` : Active rename manifest consumed by `rename-links.ps1 -Manifest`.

## Cache directories

Auto-generated by search scripts. Safe to delete (scripts re-fetch on next run).

- `accommodation_data/` : Cached Apify accommodation results.
- `car_rental_data/` : Cached Apify car rental results.
- `review_data/` : Cached Apify review results.
- `aqi_data/` : Cached OpenAQ historical air quality results.
- `lazada_data/` : Cached Apify Lazada search results.
